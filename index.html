Wallet Setup Guide
Installing MetaMask & Funding with USDC
1
Install MetaMask
Visit metamask.io and install the browser extension. Create a new wallet and save your seed phrase somewhere safe offline — never share it with anyone.
2
Add Polygon Network
In MetaMask, go to Settings → Networks → Add Network. Search for "Polygon" and add it. Polymarket runs on Polygon, not Ethereum.
3
Buy USDC on an Exchange
Create an account on Coinbase or Binance UK. Buy USDC (US Dollar Coin) with your £300. USDC is a stablecoin — 1 USDC = $1 USD at all times.
4
Send USDC to MetaMask (Polygon network)
Withdraw your USDC from the exchange to your MetaMask wallet address. Make sure to select Polygon network when withdrawing, not Ethereum — otherwise you'll pay high fees.
5
Connect to Polymarket
Visit polymarket.com and connect your MetaMask wallet. Your USDC balance will appear automatically.
6
Get Anthropic API Key
Visit console.anthropic.com, create a free account, and generate an API key. You'll need a small amount of API credits (a few dollars covers hundreds of analyses).
